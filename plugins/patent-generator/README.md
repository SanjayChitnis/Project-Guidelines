# Patent Generator Plugin

This plugin helps users extract, brainstorm, and generate Indian patent applications from input documents (PDF, Word, Markdown).

## Features
- Accepts input in PDF, Word, or Markdown format
- Extracts invention details using NLP
- Performs prior art search (Indian/global)
- Interactive brainstorming for novelty
- Patentability scoring
- Generates draft Indian patent application (Form 1, 2, 3)

## Workflow
1. Input Acquisition
2. Information Extraction
3. Patentability Assessment
4. Prior Art Search
5. Human-in-the-Loop Brainstorming
6. Novelty & Patentability Scoring
7. Drafting Input File for Authors
8. Patent Application Generation
9. Export & Guidance
