# Agent Skill Instructions: patent-prior-art

## Purpose
Guide the user through a prior art search and novelty brainstorming using AI.

## Agent Actions
- Use AI to:
  - Search for similar inventions or patents using keywords and concepts from the extracted details (simulate search using public patent databases and summarize findings).
  - Present a summary of the closest prior art and highlight overlapping features.
  - Flag any features that are not novel or are already disclosed in prior art.
- Guide the user to:
  - Discuss how their idea is different from the prior art.
  - Suggest pivots or improvements to enhance novelty.
- Document the discussion and any changes made to the invention.

## Output
- Prior art summary, flagged overlaps, and documented brainstorming session.
